# Visão Geral - InfraCare

InfraCare é uma plataforma interna para operação e governança de infraestrutura hospitalar.

## Objetivo do produto
- Monitorar ativos e serviços críticos
- Gerenciar incidentes e chamados de TI
- Executar checklists operacionais periódicos
- Consolidar indicadores de SLA
- Fornecer trilha de auditoria e relatórios gerenciais

## Escopo atual do MVP
- Backend NestJS + Prisma + PostgreSQL
- Frontend Next.js + Tailwind
- Auth JWT + RBAC
- Dashboard operacional, status page, relatórios CSV
