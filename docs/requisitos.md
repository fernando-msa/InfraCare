# Requisitos

## Funcionais
1. Autenticação e autorização por perfil
2. CRUD de ativos, incidentes e chamados
3. Execução de checklists com regra crítica
4. Dashboard com indicadores operacionais
5. Auditoria de ações relevantes
6. Exportação de relatórios em CSV

## Não funcionais
- Arquitetura modular e escalável
- Tipagem forte e validação
- Docker para ambiente local
- CI com lint/build/test
